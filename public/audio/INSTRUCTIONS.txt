HOW TO ADD YOUR CUSTOMER-AGENT VOICE RECORDING
================================================

1. RECORD OR DOWNLOAD YOUR MP3 FILE
   - Record a conversation between a customer and agent
   - Or use text-to-speech tools to generate the audio
   - Or use any sample customer service call audio

2. NAME THE FILE
   - Must be named exactly: customer-agent-conversation.mp3
   - Case sensitive!

3. PLACE THE FILE
   - Copy the MP3 file to this directory: public/audio/
   - Final path should be: public/audio/customer-agent-conversation.mp3

4. THAT'S IT!
   - The audio will automatically play when you click the play icon in the call page
   - No code changes needed

AUDIO CONTENT SUGGESTIONS:
==========================
The conversation should include common support issues like:
- Curriculum link not received
- Dashboard not opening properly
- Course content unclear
- Videos playing too fast
- Login issues
- Account problems
- Technical difficulties

RECOMMENDED TOOLS FOR CREATING AUDIO:
=====================================
Text-to-Speech:
- Google Text-to-Speech
- Amazon Polly
- Natural Reader
- TTSMaker (free online)

Recording:
- Audacity (free desktop software)
- Phone voice recorder
- Online voice recorder tools

FILE REQUIREMENTS:
==================
- Format: MP3
- Recommended duration: 1-3 minutes
- Quality: Any (but 128kbps or higher recommended)
- Size: Under 5MB recommended
